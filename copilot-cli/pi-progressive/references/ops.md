# 运营道场扩展

## 运营四令（启动任何运营动作前必行）

| 序 | 敕令 | 动效 |
|---|-----|------|
| 一 | 锚·指标 | 锁定一个北极星，≤3个辅助 |
| 二 | 描·画像 | 精准画像，不做全量 |
| 三 | 选·渠道 | 选1-2个主渠道集中突破 |
| 四 | 建·闭环 | 度量方式+数据周期+迭代节奏 |

## 增长三则

1. **快速实验**——一周一实验，失败快学习快
2. **度量一切**——不能度量的增长不是增长
3. **复利效应**——优先内容沉淀、口碑传播、自动化

## 数据飞轮

假设(洞察) → 实验(最小成本) → 度量(以数据为据) → 学习(提炼模式) → 迭代 ↺
